# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-06

### Added
- Modular `bot_sh` package with CLI, scraper, outputs, and models layers.
- Interactive CLI with match discovery, filtering, sorting, and selection.
- Spinner-based collection output with final summary.
- Basic unit tests for CLI stat mapping and output formatting.

### Changed
- Batch collection standardized and CLI orchestration consolidated.
- Summary output sorted by average (descending) for each stat.

### Removed
- Deprecated helper scripts and clickability report from tracking.
